# Blog

Minimal Hugo blog (neon-analog theme).

## Yapı
- Yazılar: `content/posts/*.md`
- Tasarım: `layouts/`
- Stil: `static/css/main.css`

## Çalıştır
- `hugo server -D --baseURL http://localhost:1313/`
- Tarayıcı: `http://localhost:1313/`

## Deploy (GitHub Pages)
- `main` branch’e push → `gh-pages` branch’e yayınlanır.
